# Видео для главного баннера

Поместите сюда видео файл `hero.mp4` для использования на главном баннере.

## Требования к видео:
- Формат: MP4
- Кодек: H.264
- Рекомендуемое разрешение: 1920x1080 (Full HD)
- Рекомендуемый размер: до 5-10 МБ (для быстрой загрузки)
- Длительность: 10-30 секунд (будет зациклено)
- Тематика: море, горы, водопады, природа Сочи/Абхазии

## Где найти видео:
1. **Pixabay** (pixabay.com) - бесплатные видео без авторских прав
2. **Videvo** (videvo.net) - бесплатные стоковые видео
3. **Coverr** (coverr.co) - бесплатные видео для веб-сайтов
4. Собственное видео, снятое вами

## Оптимизация видео:
Для уменьшения размера файла можно использовать:
- HandBrake (бесплатная программа)
- FFmpeg (командная строка)
- Онлайн-сервисы для сжатия видео

Пример команды FFmpeg для оптимизации:
```bash
ffmpeg -i input.mp4 -vcodec h264 -acodec aac -crf 23 -preset medium -movflags +faststart hero.mp4
```

После добавления файла `hero.mp4` в эту папку, видео автоматически появится на главном баннере.
