# Что можно извлечь из страницы экскурсии

## Основная информация

1. **Заголовок (H1)**
   - `<h1 itemprop="name">Долина легенд и 33 водопада.</h1>`

2. **Краткое описание**
   - `<div class="product-short-description">+ Шоу программа ансамбля «Фишт»</div>`

3. **Мета-данные**
   - `<meta name="title" content="Долина легенд и 33 водопада." />`
   - `<meta name="description" content="Долина легенд и 33 водопада. Экскурсии + Шоу программа ансамбля «Фишт`" />`
   - `<title>Экскурсии : Долина легенд и 33 водопада.</title>`

## Цены и стоимость

4. **Основная цена**
   - `<span class="PricesalesPrice">3000 руб</span>`
   - Также в JSON-LD: `"price": "3000"`

5. **Цены по точкам отправления (простой формат)**
   - `<div class="product-field-display">Отправление из Сочи - 1000 р. <br/>Отправление из Хосты - 900 р. <br/>Отправление из Адлера - 500 р.</div>`

6. **Цены по точкам отправления (с разделением на взрослых/детей)**
   - `<div class="product-field-display">Отправление из Сочи - 3000р.взр./ 2500р.дет.(7-14) <br/>Отправление из Хосты - 3000р.взр./ 2500р.дет.(7-14) <br/>Отправление из Адлера-3000р.взр./ 2500р.дет.(7-14)</div>`
   - **Формат**: `{цена}р.взр./ {цена}р.дет.(возраст)`
   - **Извлекаемые поля**:
     - `price_adult`: цена для взрослых
     - `price_child`: цена для детей
     - `child_age`: возрастные ограничения (например, "7-14")

## Временные параметры

7. **Продолжительность**
   - Простой формат: `<div class="product-field-display">Продолжительность: 10ч</div>`
   - Диапазон: `<div class="product-field-display">Продолжительность: 9-10ч</div>`

7. **Календарь доступных дат**
   - `<div id="event-calendar"></div>` (генерируется через JavaScript)
   - Скрытое поле: `<input class="vmcustom-event" type="hidden" id="vmcustom-event" name="customProductData[14][48][172][comment]">`

## Точки посадки

8. **Список точек посадки с временем**
   - В `<select>` элементе:
     ```html
     <option value="303">Лавровая - 11:00</option>
     <option value="304">Известия - 11:00</option>
     <option value="305">Знание - 11:05</option>
     ...
     ```
   - **Извлекаемые данные**:
     - Название точки посадки
     - Время отправления
     - ID опции (может использоваться для заказа)

## Дополнительные расходы

9. **Дополнительные расходы**
   - С ценами: `<div class="product-field-display">1700 р. / 1400 р. (с 5 лет до 17 лет вкл.) - переправа, концерт, экологический сбор</div>`
   - Включено в стоимость: `<div class="product-field-display">Канатная дорога на курорте «Газпром Поляна»-включена в стоимость.</div>`
   - **Важно**: Нужно различать дополнительные расходы и включенные услуги
