# Парсер сайта hostaotdykh.ru

Скрипт для парсинга всего контента с сайта https://hostaotdykh.ru/ и сохранения данных в JSON формате.

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Использование

### Первичный парсинг

```bash
python parser.py
```

Скрипт автоматически:
1. Обходит все страницы сайта
2. Извлекает текстовый контент, изображения, контакты
3. Сохраняет данные в файл `parsed_data.json` в корне проекта

### Переклассификация данных

Если данные уже собраны, но нужно улучшить классификацию страниц:

```bash
python analyze_data.py
```

Скрипт проанализирует уже собранные данные и переклассифицирует страницы на:
- Экскурсии (по ключевым словам и признакам)
- Услуги
- Контакты
- Обычные страницы

Результат сохраняется в `parsed_data_classified.json`

## Структура выходных данных

JSON файл содержит следующие разделы:

- `pages` - все обычные страницы сайта
- `excursions` - страницы экскурсий с ценами и описаниями
- `services` - страницы услуг
- `contacts` - контактная информация (телефоны, email, адреса)
- `images` - все изображения с сайта
- `navigation` - навигационные ссылки

## Особенности

- Автоматический обход всех внутренних ссылок
- Извлечение контактной информации (телефоны, email, адреса)
- Парсинг цен и длительности для экскурсий
- Сохранение метаданных страниц (title, description)
- Обработка изображений с alt-текстами
- Защита от дублирования (посещенные URL не обрабатываются повторно)
