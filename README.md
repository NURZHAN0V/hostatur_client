# Хостинский отдых г. Сочи

Современный веб-сайт туристической компании "Хостинский отдых" из Сочи. Сайт предоставляет информацию об экскурсиях, услугах трансфера, размещения и организации мероприятий.

## Технологии

- **Vue 3** - прогрессивный JavaScript фреймворк
- **Vite** - быстрый сборщик и dev-сервер
- **Vue Router** - маршрутизация
- **Pinia** - управление состоянием
- **TailwindCSS 4** - утилитарный CSS фреймворк
- **shadcn/ui** - компоненты UI на базе Radix UI
- **Lucide Vue Next** - иконки

## Структура проекта

```
src/
├── components/          # Переиспользуемые компоненты
│   ├── ui/             # UI компоненты (навигация, кнопки и т.д.)
│   ├── Layout.vue      # Основной layout
│   ├── AppHeader.vue    # Шапка сайта
│   ├── AppFooter.vue   # Подвал сайта
│   └── ...             # Другие компоненты
├── views/              # Страницы приложения
│   ├── HomeView.vue    # Главная страница
│   ├── ExcursionsView.vue
│   ├── ServicesView.vue
│   ├── ContactsView.vue
│   └── ...
├── router/             # Конфигурация роутинга
├── stores/              # Pinia stores
├── lib/                 # Утилиты
└── style.css            # Глобальные стили и цветовая палитра
```

## Цветовая палитра

Дизайн выполнен в стиле Сочи с морской тематикой:

- **Основной цвет (Primary)**: Бирюзовый (#55c4e8) - символизирует море
- **Акцент (Accent)**: Теплый оранжевый - символизирует солнце
- **Фон**: Светлые оттенки с голубыми нотками
- **Текст**: Темно-синий для контраста

## Страницы

1. **Главная** (`/`) - Hero-секция, популярные экскурсии, призыв к действию
2. **О компании** (`/about`) - Информация о компании и наградах
3. **Экскурсии** (`/excursions`) - Каталог экскурсий с фильтрацией по категориям
4. **Услуги** (`/services`) - Описание всех предоставляемых услуг
5. **Туристам** (`/tourists`) - Полезная информация для туристов
6. **Контакты** (`/contacts`) - Контактная информация и форма обратной связи
7. **Корзина** (`/cart`) - Корзина покупок

## Установка и запуск

```bash
# Установка зависимостей
yarn install

# Запуск dev-сервера
yarn dev

# Сборка для production
yarn build

# Предпросмотр production сборки
yarn preview

# Линтинг
yarn lint
```
